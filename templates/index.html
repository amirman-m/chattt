Welcome to FAME Azure API Integration

FAME powered with AI is ready to assist you in generating full automated mode.

You can send up to 4 messages. Each message will be processed by a different Azure API:

  • API 1: Language Analysis
  • API 2: Sentiment Analysis
  • API 3: Entity Recognition
  • API 4: Intent Classification

If you're satisfied with an API response, it will be used as input for the next API.
